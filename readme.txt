В проекте присутствуют директивы import с характерным для определенного проекта путем.
Во всех директивах import, начинающихся с "from Python_projects.ClientsPlanner...." следует заменить на свой путь
к указанным файлам в своем проекте.

Например если вы создали новый проект, в который положили папку ClientsPlanner из архива, то путь
"from Python_projects.ClientsPlanner...." следует изменить на "from ClientsPlanner...."

Если вы создали проект, в который выгрузили все файлы из папки ClientsPlanner из архива, то путь
"from Python_projects.ClientsPlanner...." следует изменить на "from ..."

Данные директивы есть в файле main.py и update_data.py
